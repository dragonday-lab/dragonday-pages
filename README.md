
# Dragon’s Story – GitHub Pages 패키지

이 리포지토리는 GitHub Pages로 배포할 수 있는 간단한 정적 사이트 샘플입니다.

## 구성
- `index.html`: 데모 홈 페이지
- `copyright.html`: 저작권 안내 페이지
- `assets/css/style.css`: 기본 스타일
- `.nojekyll`: GitHub Pages에서 Jekyll 처리 비활성화(선택)

## 배포 방법
1. GitHub에 새 Public 리포지토리를 만들고 파일을 업로드합니다.
2. 리포지토리 **Settings → Pages**에서 `Deploy from a branch`, `main /root`로 설정합니다.
3. 발급된 URL로 접속합니다: `https://USERNAME.github.io/REPO`.
4. 커스텀 도메인을 사용할 경우 CNAME 레코드를 `USERNAME.github.io`로 설정 후 Pages에 입력하세요.
